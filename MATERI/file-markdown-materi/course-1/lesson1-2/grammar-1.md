# This / That

## Definition

**This** dan **That** digunakan untuk menunjuk **benda tunggal (satu benda)** dalam bahasa Inggris.

Grammar ini sangat sering dipakai saat kita ingin menunjukkan atau menyebutkan benda di sekitar kita, biasanya disertai:

- Gerakan menunjuk
- Konteks jarak (dekat / jauh)

## Difference Between This and That

| Word | Meaning | Used For                          |
| ---- | ------- | --------------------------------- |
| This | Ini     | Benda yang dekat dengan pembicara |
| That | Itu     | Benda yang jauh dari pembicara    |

## Situation Examples

- Benda di tangan atau di meja dekat kita â†’ **This**
- Benda di seberang ruangan atau jauh â†’ **That**

## Sentence Pattern

**This / That + is + Noun**

<div class="note">
<strong>Penjelasan Penting:</strong>
<ul>
  <li><strong>This</strong> dan <strong>That</strong> hanya digunakan untuk <strong>benda tunggal</strong></li>
  <li>Karena subjeknya tunggal, <em>to be</em> yang digunakan <strong>SELALU is</strong></li>
  <li>Jangan gunakan <strong>are</strong>, karena <strong>are</strong> hanya untuk benda jamak</li>
</ul>
</div>

## Example Sentences

- This is a book. _(Ini sebuah buku)_
- This is my bag. _(Ini tas saya)_
- That is a house. _(Itu rumah)_
- That is a chair. _(Itu kursi)_
