# Possessive Adjective (My)

## Definition

**Possessive adjective** digunakan untuk menunjukkan **kepemilikan** (siapa pemilik suatu benda).

Kata **my** berarti _"milik saya"_ dan **selalu digunakan sebelum kata benda (noun)**.

Grammar ini menjawab pertanyaan:  
**"Ini punya siapa?"**

## Sentence Pattern

**My + Noun**

<div class="note">
<strong>Penjelasan Penting:</strong>
<ul>
  <li><strong>My</strong> <strong>tidak pernah</strong> berdiri sendiri</li>
  <li><strong>My</strong> <strong>selalu diikuti</strong> oleh kata benda</li>
  <li>Jangan menambahkan <strong>is / are</strong> setelah <strong>my</strong></li>
</ul>
</div>

## Example Usage

- My bag _(tas saya)_
- My phone _(handphone saya)_
- My house _(rumah saya)_
- My book _(buku saya)_

## Example Sentences

- This is my bag. _(Ini tas saya)_
- That is my phone. _(Itu HP saya)_
