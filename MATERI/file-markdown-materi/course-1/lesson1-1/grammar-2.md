# Introducing Yourself (My name is)

## Definition
Pola **My name is** digunakan untuk memperkenalkan nama diri secara sederhana dan sopan dalam bahasa Inggris.

Biasanya digunakan saat:
- Pertama kali bertemu seseorang
- Menyapa orang baru

Pola ini **sangat umum** dipakai dalam percakapan sehari-hari dan **cocok untuk pemula**.

## Sentence Pattern
**My name is + Name**

<div class="note">
<strong>Catatan:</strong>
<ul>
  <li><strong>"My"</strong> berarti <em>"saya punya"</em></li>
  <li><strong>"name"</strong> berarti <em>"nama"</em></li>
  <li><strong>"is"</strong> adalah bentuk <em>to be</em> yang digunakan karena subjeknya <strong>"name"</strong> (tunggal)</li>
</ul>
</div>

## Example Sentences
- My name is Alex.
- My name is Rose.

## Example in Conversation
- Hello, my name is Alex.
- Hi, my name is Rose. Nice to meet you.
