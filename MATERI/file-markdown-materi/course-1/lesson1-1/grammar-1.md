# To Be (am, is, are)

## Definition
**To be** adalah kata kerja penghubung (*linking verb*) yang digunakan dalam bahasa Inggris untuk menghubungkan subjek dengan:

- Identitas (siapa seseorang atau apa sesuatu)
- Keadaan atau sifat (*adjective*)
- Profesi atau peran

Dalam bahasa Indonesia, kata ini sering tidak terlihat secara langsung  
(contoh: *"Saya senang"*), tetapi dalam bahasa Inggris, **to be WAJIB digunakan** agar kalimat benar secara tata bahasa.

## Forms of To Be
| Subject | To Be |
|--------|-------|
| I      | am    |
| You    | are   |
| He / She / It | is |

<div class="note">
<strong>Catatan Penting:</strong>
<ul>
  <li><strong>I</strong> selalu menggunakan <strong>am</strong></li>
  <li><strong>You</strong> selalu menggunakan <strong>are</strong>, baik tunggal maupun jamak</li>
  <li><strong>He / She / It</strong> selalu menggunakan <strong>is</strong></li>
</ul>
</div>
