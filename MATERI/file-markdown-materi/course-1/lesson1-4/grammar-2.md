# Describing Nouns (Simple)

## Definition

Grammar ini digunakan untuk menjelaskan **kata benda (noun)** dengan menggunakan **adjective**.

Berbeda dengan grammar sebelumnya (_Adjective with To Be_), di sini **adjective langsung ditempatkan sebelum noun**.

## Sentence Pattern

**Adjective + Noun**

## Example Phrases

- a big house _(rumah besar)_
- a smart student _(murid pintar)_
- a small bag _(tas kecil)_

## Example Sentences

- This is a big house.
- She is a smart student.
