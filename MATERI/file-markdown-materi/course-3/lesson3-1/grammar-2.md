# For and Since

## Definition

**For** dan **Since** digunakan untuk menyatakan **durasi waktu** dalam **Present Perfect Tense**.

Keduanya sama-sama berbicara tentang **berapa lama**, tetapi cara penggunaannya **berbeda**.

## For

**For** digunakan untuk menunjukkan **jumlah waktu / durasi**.

Digunakan dengan:

- for 2 years
- for a long time
- for three months

## Since

**Since** digunakan untuk menunjukkan **titik awal waktu**.

Digunakan dengan:

- since 1990
- since last October
- since Monday
- since I was born

## Example Sentences

- I have lived here for 10 years.
- She has worked here since last October.
