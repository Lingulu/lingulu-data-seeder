# Second Conditional Structure

## Definition

**Second Conditional** digunakan untuk berandai-andai tentang **situasi yang tidak nyata** atau **sangat kecil kemungkinannya terjadi saat ini** (Present / Future).

Contoh makna:

> _“Jika saya punya sayap (faktanya tidak punya), saya akan terbang.”_

## Sentence Pattern

**If + Subject + Verb 2 (Simple Past), Subject + Would + Verb 1 (Base Form)**

## Important Explanation

Meskipun menggunakan **Verb 2 (Past)**, maknanya adalah **SEKARANG atau masa depan**, **bukan masa lalu**.

## Example Sentence

- If I had money, I would buy a car.  
  _(Fakta: I don't have money now)_
