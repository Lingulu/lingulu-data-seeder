# "If I were you"

## Definition

Dalam **Second Conditional**, bentuk **to be** untuk semua subjek  
(**I, He, She, It**) adalah **were**, **bukan was**.

Ini disebut bentuk **formal** dalam bahasa Inggris dan dikenal sebagai **subjunctive mood**.

Frasa **"If I were you"** sangat sering digunakan untuk **memberi saran** kepada orang lain.

## Key Rule

- Gunakan **were** untuk semua subjek dalam conditional type 2
- Jangan gunakan **was** pada struktur ini

## Example Sentences

- If I were you, I would accept the offer.  
  _(Jika saya jadi kamu...)_

- If she were here, she would be happy.
