# The Agent (By ...)

## Definition

Dalam **kalimat pasif**, pelaku aksi disebut **agent**.

Kita menggunakan kata **by** untuk menyebutkan agent tersebut.  
Namun, jika pelakunya **tidak diketahui** atau **tidak penting**, bagian **by + agent** bisa **dihilangkan**.

## Usage Rules

- Gunakan **by + agent** jika **siapa pelakunya penting**
- Hilangkan **by + agent** jika pelaku:
  - tidak diketahui, atau
  - tidak relevan dengan makna kalimat

## Example Sentences

- The book was written by J.K. Rowling.  
  _(Penting siapa penulisnya)_

- My wallet was stolen.  
  _(Tidak diketahui siapa pencurinya, jadi tidak menggunakan "by")_
