# Must Be (Strong Certainty)

## Definition

Kita menggunakan **"must be"** ketika kita **sangat yakin** bahwa sesuatu itu benar berdasarkan **bukti yang logis**  
(tingkat keyakinan sekitar **90–100%**).

Dalam konteks ini, **must be bukan berarti “harus”**, tetapi berarti **“pasti”**.

## Sentence Pattern

**Subject + Must Be + Adjective / Noun**

## Example Sentence

- You have been working all day. You must be tired.  
  _(Logika: kerja seharian → pasti capek)_
