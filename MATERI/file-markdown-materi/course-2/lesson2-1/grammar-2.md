# Past To Be (Was / Were)

## Definition

Jika kalimat **tidak memiliki kata kerja (aksi)**, tetapi menjelaskan **keadaan, lokasi, atau sifat di masa lalu**, maka digunakan **was** atau **were**.

## Subject Pairing

- **I / He / She / It** → **Was**
- **You / We / They** → **Were**

## Sentence Pattern

**Subject + Was / Were + Adjective / Noun**

## Example Sentences

- I was tired last night.  
  _(Saya lelah tadi malam)_

- They were busy yesterday.  
  _(Mereka sibuk kemarin)_
