# Comparative (Comparing Two Things)

## Definition

**Comparative** digunakan untuk **membandingkan sifat** antara **dua benda atau dua orang**.

## Rules

### 1. Short Adjectives (1 syllable)

Untuk kata sifat **pendek (1 suku kata)**, tambahkan **-er + than**.

- Fast → Faster than
- Big → Bigger than

### 2. Long Adjectives (2 syllables or more)

Untuk kata sifat **panjang (2 suku kata atau lebih)**, gunakan **more + adjective + than**.

- Expensive → More expensive than
- Beautiful → More beautiful than

### 3. Irregular Adjectives

Beberapa adjective memiliki bentuk perbandingan **tidak beraturan** dan harus dihafal.

- Good → Better
- Bad → Worse

## Example Sentences

- This car is faster than that bus.
- The bag is more expensive than the shoes.
