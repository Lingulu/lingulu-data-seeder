# Will (Spontaneous / Prediction)

## Definition

Kata **will** digunakan untuk membicarakan **masa depan**, tetapi secara spesifik dipakai untuk:

- Keputusan **mendadak** saat berbicara (spontan)
- **Prediksi** tanpa bukti kuat
- **Janji**

## Usage Explanation

Will biasanya digunakan ketika keputusan **baru dibuat saat itu juga**, bukan direncanakan sebelumnya.

Untuk prediksi, will dipakai jika prediksi tersebut berdasarkan **opini atau perasaan**, bukan bukti yang jelas.

## Sentence Pattern

**Subject + Will + Verb 1 (Base Form)**

## Example Sentences

- It is hot here. I will open the window.  
  _(Keputusan spontan)_

- I think it will rain tomorrow.  
  _(Prediksi tanpa bukti kuat)_
