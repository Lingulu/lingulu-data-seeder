# Spelling Rules for V-ing

## Definition

Saat mengubah **Verb 1** menjadi bentuk **V-ing**, ada beberapa **aturan ejaan** yang perlu diperhatikan.

## Spelling Rules

### 1. Final "e" is removed

Jika kata kerja berakhiran **-e**, maka huruf **e biasanya dihilangkan**.

- Write → Writing _(bukan Writeing)_

### 2. Consonant–Vowel–Consonant (CVC)

Jika kata kerja berakhiran **konsonan–vokal–konsonan** dengan bunyi pendek, maka **konsonan terakhir didouble**.

- Run → Running
- Sit → Sitting

### 3. Normal Form

Jika tidak termasuk aturan di atas, cukup tambahkan **-ing**.

- Read → Reading
- Play → Playing

## Example Sentences

- He is writing an email.
- They are running in the park.
