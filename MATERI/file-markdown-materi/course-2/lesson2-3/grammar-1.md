# Present Continuous Tense

## Definition

**Present Continuous Tense** digunakan untuk menjelaskan **aktivitas yang sedang berlangsung** tepat saat kita berbicara, atau untuk **situasi sementara**.

## Sentence Pattern

**Subject + To Be (am / is / are) + Verb-ing**

## Example Sentences

- I am eating lunch right now.  
  _(Saya sedang makan siang sekarang)_

- She is working on a new project.
